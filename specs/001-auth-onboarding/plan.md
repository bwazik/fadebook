## Summary

Implement the foundational user identity and shop onboarding flows for FadeBook. This includes client and shop owner registration via Egyptian phone numbers, role-based login, a multi-step Livewire onboarding wizard for barbershops, an admin approval/rejection cycle for shops, and a WhatsApp-based OTP password reset flow.

## Technical Context

**Language/Version**: PHP 8.5+ with Laravel 13
**Primary Dependencies**: Livewire 4, Alpine.js, Tailwind CSS v4, Filament 5 (implied integration)
**Storage**: MySQL 8+
**Testing**: Pest 4
**Target Platform**: PWA (Mobile-first web app targeting iOS Safari & Android Chrome)
**Project Type**: Web Application
**Performance Goals**: < 60s for registration, < 15s for login, < 2m for password reset
**Constraints**: All layout must be RTL, text in Egyptian Arabic. SMS/Email disabled; uses WhatsApp stub. No MySQL ENUM types.
**Scale/Scope**: Phase 1 MVP features covering 7 core user stories for authentication, role authorization, and shop details capture.
